Gonk is my attempt at recreating Projectyle - an old Atari/Amiga game - for the PC.
A three-player game of air-hockey-style action. This is an early release and there's not much information yet.
Put the files in a directory and run Gonk.exe. Note: I now have Gonk running under Mac OS X. I need to do some tuning and figure out the proper place to store the config file. It'll be available for download soon.

Magus is a turn-based RPG...sort of. Tile based graphics and some inappropriate sound fx.
This is none the less a very entertaining game, particulary for more than one player!
(requires a mouse but doesn't check for it)
(Pentium Pro users: runs in mode 12h...you may need to disable Fastvid)
