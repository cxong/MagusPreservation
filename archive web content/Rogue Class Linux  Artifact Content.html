Introduction
============
Magus is a turn-based RPG with tile-based graphics.  It is one of the
few local multi-player cooperative roguelikes.  It supports multiple
characters in the party.  Simply create multiple characters before
starting the game.  It requires a mouse, but it can be played with a
joystick and the virtual mouse.

How to play
===========
At the welcome screen, press any key.

At the character creation screen, use the right and left arrow keys,
or the << and >> buttons to select a class.  Type in the character's
name.  Press Enter or click OK to continue.  Adjust the statistics as
desired.  Press Enter or click OK to continue.  This will return you
to the character creation screen where you may create an additional
character.  When finished creating the party, click Done.

Every character has a certain number of movement points (MP) per turn.
Use the < and > buttons to switch between characters at any time.  When
a character's MP reaches 0, they cannot perform any more actions until
the next turn.  Use the End button to end the turn for the whole party.

To pick up an item, walk over it and click the character.
To drop an item, shift-click the item in inventory.
To identify an item, left-click the item in inventory.
To equip an item, right-click the item in inventory.
To remove an item, right-click on it.
To attack a monster, right-click on the monster.
To cast a spell, equip a spell item, then right-click on the target.

The blue bar represents magic points.  The green bar represents health
points.  You can regenerate these by repeatedly ending the party's turn.

It is a good idea to equip your armor, weapons, and spells before
leaving the starting area.

For more help, see readme-v2.txt in the documentation.  The game is the
3rd edition, but it has no documentation, so the readme is from the 2nd
edition.

Magus saved games are stored in /c/magus/SAVED.DAT

Joystick controls
=================
To enable or disable the virtual mouse, press Right + Left on the
joystick.  It can be useful to disable the virtual mouse in the
character creation screen to type in a character name on the virtual
keyboard.

For more details about the virtual mouse, see the RCL User Guide.
To move the mouse quickly, press Right + a direction.
To end the whole party's turn, press Right + B.
To switch characters, press Left + West or East.
To save the mouse pointer, press Left + Select.
To restore the mouse pointer, press Left + Start.
To left-click, press B.
To right-click, press A.

Wish list
=========
Features that would be nice to have:
* More documentation
* More keyboard bindings
* A "follow" feature to assist party travel
* An "exchange" feature to quickly swap melee and ranged weapons
* Beavers as a playable race
