Magus is a turn-based RPG with tile-based graphics. It is one of the few local multi-player cooperative roguelikes. It supports multiple characters in the party. Simply create multiple characters before starting the game.

Magus has a different design than most rogue-like games and this affects the game play. New players are encouraged to read the readme files. Keep in mind that this game was designed to be played with a mouse and multiple players. The turn and movement system is based on table-top gaming. Also remember that it is the player's responsibility to equip the character's starting items.

Magus saved games are stored in /c/magus/saved.dat

You can find more information about magus at an archived copy of the author's site
